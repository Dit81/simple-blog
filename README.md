#这是基于 CodeIgniter 的简单 Blog 程序  
CodeIgniter 官网的视频教程栏目里有一个“20分钟开发一个博客”的教程，它使用控制器和视图来做，没有遵从规范，省去了模型的部分。我在这里加进了模型，用“模型--控制器--视图”的完整架构来开发。都是简单的一些数据库操作和内容输出，CSS 和 js 内容近乎没有，显得简单。这是一个学习过程。   
我会慢慢加进一些额外的功能，让它显得比较完整。接下来，我想通过 SESSION 的技术写个用户登录检验的功能。有好消息时告诉你。页面视图分离了 header footer，在 header 加入 $this->session->userdata['id'] 验证，若未登录将被强制到登录页面。   
加进多用户写作功能，每个用户可以写作发布自己的博文，登录到首页时显示的也是所属自己的博文。这是通过更新数据库语句来达成，查询时加多一个限制条件，插入时加多一个 author_id 值。
